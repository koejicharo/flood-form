—

📌 วิธีใช้งานสำหรับผู้ดูแล (Admin)

  1. กรอก ยศ ชื่อ สกุล ตำแหน่ง สังกัด ของเจ้าหน้าที่แต่ละคน
  2. กด "สร้าง Link" จะได้ Link เฉพาะของเจ้าหน้าที่คนนั้น
  3. ส่ง Link ผ่าน LINE / WhatsApp / SMS ให้เจ้าหน้าที่
  4. เจ้าหน้าที่เปิด Link → กรอกข้อมูลส่วนที่เหลือ → กดพิมพ์เป็น PDF
  5. ทำซ้ำสำหรับแต่ละคน (ไม่ต้องติดตั้งโปรแกรม)
